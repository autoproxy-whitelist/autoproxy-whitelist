autoproxy_whitelist
===================
想必大家都知道有一个著名的autoproxy_gfwlist

但是，那个列表只记录了一些被墙的网址，而且很不全面，所以出现了对白名单的需求
本项目正是为此而生
===================
我们使用和gfwlist相同的格式为大家呈现这个列表
